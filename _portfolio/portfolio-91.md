This project showcases a **web-based SRT Live Streaming Dashboard** developed to receive, monitor, and manage live video streams using the **Secure Reliable Transport (SRT)** protocol.

The system serves as a lightweight browser-accessible dashboard that allows operators to receive live feeds from Android streaming applications and professional camera encoders in real time.

Designed for broadcasting environments, the application provides an easy-to-use interface for monitoring active streams with low latency and reliable transmission.

---

# 🌐Web Application Overview

This web application functions as a centralized monitoring dashboard for incoming SRT streams.

Supported streaming sources include:

- 📱 Android Live Streaming Application
- 🎥 Professional Camera Encoder (SRT)

Unlike the desktop server version, this web application **does not support direct Blackmagic Fiber Optic (FO) input**.

# 🔄 System Workflow

1. Open Web Dashboard
2. Configure Receiver
3. Android App or Camera Encoder Sends SRT Stream
4. Server Receives Incoming Stream
5. Web Dashboard Displays Live Video
6. Operator Monitors Stream Status
