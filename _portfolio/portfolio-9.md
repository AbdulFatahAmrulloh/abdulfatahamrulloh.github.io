This project showcases a **professional desktop-based SRT Live Server** developed for Windows. The application serves as a centralized **SRT (Secure Reliable Transport) receiving server**, capable of accepting live video streams from multiple broadcasting sources while providing reliable, low-latency transmission for professional production environments.

Unlike a mobile broadcasting application, this software functions as the **central receiving endpoint** in the streaming workflow. It receives incoming SRT streams from Android applications, professional hardware encoders, and broadcast equipment before distributing them to downstream production systems.

The application has been successfully implemented in a real-world broadcasting environment, demonstrating high reliability and stable performance for continuous live production.

---

# 🖥Desktop Application Overview

This desktop application acts as a professional SRT receiving server capable of managing multiple incoming live streams simultaneously.

It supports several broadcasting sources including:

- 📱 Android Live Streaming Application
- 🎥 Kiloview Professional SRT Encoder
- 🎬 Blackmagic Broadcast Devices
- 🌐 Other SRT-compatible encoders

The software centralizes all incoming streams into a single server, simplifying broadcast infrastructure and improving production efficiency.
